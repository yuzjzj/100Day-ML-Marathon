## 資料同步(Git設定)
### 初始設定
```sh
git config --global user.name "你的名字 中間有空格記得加雙引號"
git config --global user.email "你的E-mail"
```

cd /windows/workspace/100Day-ML-Marathon/

檔案(下載)：

`git pull`

檔案推送(上傳)：
```sh
git add Day_001_HW.ipynb
git commit -m "Day001上傳"
git push
```
資料比對：
```sh
git diff
git status
```
